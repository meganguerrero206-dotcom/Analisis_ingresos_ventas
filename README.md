# 📊 Dashboard de Análisis de Ingresos y Ventas - Veterinaria

Este proyecto consiste en un dashboard interactivo desarrollado en Power BI para analizar el rendimiento financiero, las ventas de productos/servicios y las tendencias de ingresos en una clínica veterinaria.

<img width="1464" height="814" alt="Dashboard_principal png" src="https://github.com/user-attachments/assets/270671d9-358c-46ac-92a1-3b45c04783fe" />

## 🧠 ¿Qué incluye este repositorio?
* `Analisis_Ingresos_Veterinaria.pbix`: Archivo ejecutable de Power BI con el modelo de datos y los reportes visuales.

## 📈 Insights Clave del Negocio (Basados en el Reporte)
* **Ingresos Totales:** Se alcanzó un total de **$15,893,493** en ingresos acumulados.
* **Canales de Mayor Impacto:** 
  * Los **Gatos** lideran los ingresos por tipo de animal con **$3.9 mill.**
  * El servicio de **Emergencia** es el más rentable, aportando **$4.2 mill.**
  * El desglose de ingresos por mascotas **Esterilizadas (49.36%)** vs **No Esterilizadas (50.64%)** se mantiene equitativo.
* **Rendimiento del Personal:** El **Dr. Conley** ($3.6 mill.) y el **Dr. Price** ($3.5 mill.) son los médicos con mayor facturación.

## 🛠️ Tecnologías y habilidades utilizadas
* **Power BI Desktop** para el modelado de datos y diseño de la interfaz (Dark Mode).
* **Power Query** para el proceso de extracción, transformación y limpieza de datos (ETL).
* **DAX (Data Analysis Expressions)** para la creación de métricas clave (ingresos totales, segmentación por servicios y médicos).
